# Machine Learning Based Stock Portfolio Risk Analyzer with Trend Prediction and Investment Recommendation System

# Description
This project analyzes stock portfolio risk and predicts next-day stock trends using a Random Forest machine learning model. It evaluates volatility, correlation, diversification, and sector concentration to generate intelligent investment recommendations.

# Features
1 Stock trend prediction
2 Portfolio volatility analysis
3 Correlation detection
4 Diversification scoring
5 Sector concentration detection
6 Risk score (0–100)
7 Investment recommendation system

# Technologies Used
Python, Pandas, NumPy, Matplotlib, Scikit-learn, Yahoo Finance API
